# Kubernetes

Проверки здоровья кластера, ресурсов, образов. В том числе скрипты с ограниченным (read-only) доступом и для нескольких кластеров.

| Скрипт | Описание |
|--------|----------|
| *(план)* | Здоровье кластера: ноды, поды, события |
| *(план)* | Сводка по ресурсам по namespace |
| *(план)* | Поиск образов без тега / latest |
| *(план)* | Read-only проверки |
| *(план)* | Мульти-кластер: контексты, проверка нескольких кластеров |
