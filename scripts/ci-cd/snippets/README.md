# Snippets для .gitlab-ci.yml

Переиспользуемые фрагменты. Копируйте в свой pipeline или подключайте через `include`.

| Файл | Назначение |
|------|------------|
| `snippet-run-python.md` | Как запустить Python-скрипт из репозитория в job. |
| `snippet-artifacts-reports.md` | Сохранение текстовых отчётов в артефакты. |
| `snippet-scheduled-job.md` | Job только по расписанию (schedule). |
| `snippet-helm-upgrade.md` | Один job: helm upgrade --install. |
| `snippet-secrets-env.md` | Использование секретов через CI/CD Variables. |

Файлы в формате .md содержат готовый YAML и краткие пояснения.
