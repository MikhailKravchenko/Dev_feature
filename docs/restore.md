# Восстановление из бэкапов

Сценарии восстановления для каждой СУБД.

**Скрипты:** [../scripts/restore/](../scripts/restore/)

---

## СУБД

| СУБД | Действия |
|------|----------|
| PostgreSQL | pg_restore из custom/directory; при необходимости создание БД |
| MySQL / MariaDB | Восстановление из dump (mysql / source) |
| MongoDB | mongorestore |
| Redis | Подмена RDB-файла, перезапуск сервиса |

---

*Предупреждения и пошаговые инструкции будут добавлены при реализации.*
