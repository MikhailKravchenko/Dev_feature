# Kubernetes

Скрипты для работы с одним или несколькими кластерами. Включая варианты с **ограниченным (read-only) доступом**.

**Скрипты:** [../scripts/kubernetes/](../scripts/kubernetes/)

---

## Разделы

- **Здоровье кластера** — ноды, поды, события, недавние рестарты.
- **Ресурсы** — requests/limits, использование по namespace.
- **Образы** — поиск подов с образами без тега или с `latest`.
- **Read-only** — только get/list, без изменений ресурсов.
- **Мульти-кластер** — переключение контекста (kubectl config use-context), проверка нескольких кластеров подряд.

---

*Требования: kubectl, настроенный kubeconfig. Для read-only — достаточно роли с get/list.*
