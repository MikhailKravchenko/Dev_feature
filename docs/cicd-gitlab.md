# GitLab CI/CD

Примеры пайплайнов и jobs. Без интеграции с мониторингом.

**Примеры и snippets:** [../scripts/ci-cd/](../scripts/ci-cd/)

---

## Разделы

- **Диагностика** — job для запуска скриптов диагностики, сохранение отчётов как артефактов.
- **Бэкапы по расписанию** — scheduled pipeline для запуска скриптов бэкапа СУБД.
- **Деплой Helm** — пример deploy job для dev/stage/prod (переменные окружения, окружение в GitLab).
- **Snippets** — переиспользуемые фрагменты `.gitlab-ci.yml` с комментариями (включать через include или копировать).

---

*Секреты и учётные данные — через CI/CD Variables (masked), не в коде.*
