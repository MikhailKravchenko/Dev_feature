# Helm

Валидация чартов, сравнение с кластером, шаблоны values.

**Скрипты:** [../scripts/helm/](../scripts/helm/)

---

## Разделы

- **Валидация** — `helm lint` для выбранных чартов (локально или из репозитория).
- **Diff** — сравнение текущего состояния в кластере с тем, что получится после `helm upgrade` (например, helm diff plugin).
- **Шаблоны values** — примеры values для окружений dev/stage/prod и описание отличий.

---

*Требования: helm, при необходимости helm-diff plugin.*
