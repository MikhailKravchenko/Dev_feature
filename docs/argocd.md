# Argo CD

Скрипты для проверки статуса приложений Argo CD.

**Скрипты:** [../scripts/argocd/](../scripts/argocd/)

---

## Разделы

- **Список приложений** — имя, статус синхронизации (Synced/OutOfSync), здоровье (Healthy/Degraded/Progressing).
- **Out-of-sync и ошибки** — фильтрация приложений с расхождениями или с ошибками для последующего разбора.

---

*Требования: argocd CLI, доступ к Argo CD API (логин/токен).*
